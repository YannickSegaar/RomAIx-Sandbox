Voiceflow Webchat API Demo

How to Use This API

Send a POST request to /api/set-payload with your custom payload:


POST http://localhost:3000/api/set-payload
Content-Type: application/json

{
    "n8n_executionID": "your-execution-id",
    "n8n_resumeURL": "your-resume-url"
}

After sending the payload, you'll receive a response with a launch URL:


{
    "success": true,
    "message": "Payload received",
    "launchUrl": "http://localhost:3000/launch-chat"
}

Open the provided launchUrl to start the Voiceflow chat with your custom payload.

Example with cURL


curl -X POST http://localhost:3000/api/set-payload \
    -H "Content-Type: application/json" \
    -d '{
        "n8n_executionID": "dynamic-id-123",
        "n8n_resumeURL": "https://example.com/resume/456"
    }'
